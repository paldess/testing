#### База данных доступна по указанным в проекте данным. 



Структура управления ограничениями:

каждый пользователь относится к определенной группе. группа имеет доступ к определенным страницам сайта. через API можно изменить какая группа к каким страницам будем иметь доступ.



API аутентификация происходит по ключу, который храниться в БД и должен иметь пользователь, с правами администратора.
